# PhishGuard AI
A phishing email detection tool powered by LLaMA 3 via Groq API.

## Setup
1. Clone the repo
2. pip install -r requirements.txt
3. Set GROQ_API_KEY environment variable
4. python app.py

## Features
- Single email analysis
- Inbox scanner via IMAP
- Model benchmarking
